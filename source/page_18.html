Android 4.4创建客制化项目步骤

 

1、选择客制化项目的基础项目，即亿道公版项目，本文中以em_t8270_nand为例

2、复制以下文件及文件夹并重命名为新项目名称，本文中以“NEW_PROJECT”为例

$ cp bootable/bootloader/lk/project/em_t8270_nand.mk    bootable/bootloader/lk/project/NEW_PROJECT.mk

$ cp build/target/product/em_t8270_nand.mk  build/target/product/NEW_PROJECT.mk

$ cp build/target/product/security/em_t8270_nand/  build/target/product/security/NEW_PROJECT  -rf

$ cp mediatek/binary/packages/em_t8270_nand/  mediatek/binary/packages/NEW_PROJECT  -rf

$ cp mediatek/config/em_t8270_nand/  mediatek/config/NEW_PROJECT  -rf

$ cp mediatek/custom/em_t8270_nand/  mediatek/custom/NEW_PROJECT  -rf

$ cp vendor/mediatek/em_t8270_nand/  vendor/mediatek/NEW_PROJECT   -rf

$ mv vendor/mediatek/NEW_PROJECT/artifacts/out/target/product/em_t8270_nand/ vendor/mediatek/NEW_PROJECT/artifacts/out/target/product/NEW_PROJECT

 

3、如果包含如下文件

mediatek/custom/NEW_PROJECT/kernel/Makefile

 

则文件内容

include $(MTK_PATH_BUILD)/common.mk
$(call all-modules-src-or-makefile,$(obj),CUSTOM_KERNEL_)
obj-n := dummy.o

ifeq ($(MTK_ALPS_BOX_SUPPORT), yes)
ccflags-y += -DMTK_ALPS_BOX_SUPPORT
endif

ifeq ($(X680),yes)
obj-y += gsl_point_id.o
$(obj)/gsl_point_id.o: $(srctree)/../out/target/product/em_t8270_nan/obj/CUSTGEN/custom/kernel/touchpanel/gsl_point_id
        cp $(srctree)/../out/target/product/em_t8270_nand/obj/CUSTGEN/custom/kernel/touchpanel/gsl_point_id $(obj)/gsl_point_id.o
endif

 

修改为：

include $(MTK_PATH_BUILD)/common.mk
$(call all-modules-src-or-makefile,$(obj),CUSTOM_KERNEL_)
obj-n := dummy.o

ifeq ($(MTK_ALPS_BOX_SUPPORT), yes)
ccflags-y += -DMTK_ALPS_BOX_SUPPORT
endif

ifeq ($(X680),yes)
obj-y += gsl_point_id.o
$(obj)/gsl_point_id.o: $(srctree)/../out/target/product/NEW_PROJECT /obj/CUSTGEN/custom/kernel/touchpanel/gsl_point_id
        cp $(srctree)/../out/target/product/NEW_PROJECT /obj/CUSTGEN/custom/kernel/touchpanel/gsl_point_id $(obj)/gsl_point_id.o
endif

$./create_project.sh
